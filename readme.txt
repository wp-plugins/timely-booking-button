Adds a "Book Now" button for the Timely appointment management system to your WordPress site

== Installation ==

1. Upload the `timely-booking-button` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the widget via the Appearance > Widgets admin menu or place `&lt;?php do_action('tbb_widget_output()'); ?&gt;` in your templates
